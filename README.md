# assignment-for-nested-tags

A small React + TypeScript app (Vite) that demonstrates nested tag/tree editing with editable tag names, data nodes, and child management.

## Features
- Editable tag names
- Data nodes with inline editing
- Add children to parent nodes
- Collapse/expand nested tags
- Type-safe model with DataNode / ParentNode / TreeNode types

## Quick start

1. Install dependencies
```sh
npm install

2. Start server
npm run dev
